WKWebView Plugin Demo

The WKWebView plugin aims to be a drop-in replacement for the default iOS UIWebView. This demo is only a non-scientific comparison of WebGL performance between those webviews.

Make sure you check all features of your app before moving to production as the WKWebView is quite new and not all other APIs and plugins may be fully compatibile.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS - App package.

A little WebGL performance test

The demo in the WKWebView iframe should reach 60 FPS without eating the entire CPU. The InAppBrowser UIWebView should perform a lot worse. By the way, the difference in resolution has only marginal impact.
